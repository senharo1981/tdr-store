
# TDR-STORE Pakistan

A premium grocery delivery app designed for the local Pakistani market.

## How to use (For Beginners):
1. Create a GitHub account.
2. Create a new repository.
3. Upload `index.html` and `index.tsx`.
4. Connect to Vercel.com for a live link.

## Features:
- English & Urdu support.
- WhatsApp Order integration.
- GPS Location pin for fast delivery.
- Mobile Responsive Design.
- Premium Aesthetics.
